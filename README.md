## 🚀 **DEPLOY COMMANDS**

### **React + Vite**
```
# Development
npm run dev

# Production Build
npm run build

# Deploy Options:
# 1. Vercel: vercel --prod
# 2. Netlify: netlify deploy --prod --dir=dist
# 3. GitHub Pages: npm run deploy
```

### **HTML/CSS (Instant)**
```
# GitHub Pages
1. New Repo: /Ultra-Premium-Portfolio
2. Upload index.html
3. Settings → Pages → Deploy
4. Live: https://dev-moe-kyawaung.github.io/Ultra-Premium-Portfolio/

# Netlify
1. app.netlify.com/drop → Drag index.html
2. Instant Live URL
```
